## Duel Masters Deck Parser for OCTGN
**Very basic parsing of Duel Masters cards with multiline support. Can match by part of the text. The parser always prioritizes TCG cards to avoid adding OCG proxies.**

## Installation Guide:
![GitHub release](https://img.shields.io/github/v/release/szefo09/Duel-Masters-Deck-Parser-OCTGN?label=Latest%20Release)
- Go to the [Releases page](https://github.com/szefo09/Duel-Masters-Deck-Parser-OCTGN/releases)
- Download **Octgn.DuelMastersDeckParser.dll**
- Go to your OCTGN Data Directory like so: `OCTGN\Data\Plugins`
- Create a folder called `DuelMastersDeckParser` in `Plugins` folder.
- Copy and Paste the **Octgn.DuelMastersDeckParser.dll** into `DuelMastersDeckParser` folder.
- Launch OCTGN and go to Deck Editor -> Plugins to find the Duel Masters Deck Parser.
